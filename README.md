## 微信小程序部署

### 方式一：WebView 嵌入（推荐，最快）

1. 把 `index.html` 放到你们的服务器上
2. 在微信小程序中用 `web-view` 组件加载这个链接
3. 需要已备案的域名

### 方式二：转换为原生微信小游戏

1. 注册微信公众平台 → 小程序账号
2. 下载微信开发者工具
3. 新建项目，选择「小游戏」
4. 将 `index.html` 中的游戏逻辑迁移到 Cocos Creator
5. 接入微信广告 SDK（下方说明）

### 方式三：H5上线（最简单）

1. 把 `index.html` 部署到已备案域名
2. 微信内直接打开链接即可玩
3. 朋友圈/群分享引流

## 广告接入（变现）

目前代码里广告位已预留，调用 `simulateAd()` 的地方接真实广告 SDK：

| 广告位 | 代码位置 | 类型 |
|--------|---------|------|
| 看广告复活 | `failOverlay` 按钮 | 激励视频 |
| 双倍金币 | `completeOverlay` 按钮 | 激励视频 |
| 解锁家具 | `unlockWithAd()` | 激励视频 |
| 看广告得金币 | `showAd()` | 激励视频 |

### 微信广告接入步骤：

1. 开通「微信小程序流量主」
2. 在小程序后台创建广告位
3. 接入微信广告 SDK：
```javascript
// 激励视频广告
wx.createRewardedVideoAd({
  adUnitId: '你们的广告位ID'
})
```
